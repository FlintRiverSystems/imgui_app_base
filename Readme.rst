================================================================================
WASM build
================================================================================

Install emsdk::

   cd $FRSTPSRC
   git clone https://github.com/emscripten-core/emsdk
   cd emsdk
   ./emsdk install latest
   ./emsdk activate latest

Run ``./build_wasm.sh`` in the app_example folder
